Provides a widget to display news from the Surfrider Foundation National Headquarters 

== Description ==

The News From Surfrider National plugin provides a Wordpress Widget that displays news from Surfrider Foundation's national headquarters. 

== Installation ==

1. Upload and activate the plugin (newsfromsurfridernational.zip)
1. Drag the News From HQ widget to the place in the theme that you would like the widget to appear

== Frequently Asked Questions ==

= Who controls the news feed? =

The news feed is curated by the Surfrider Foundation's Marketing and Communications staff

= What happens if content is unavailable? =

The Widget uses AJAX to load the news content asynchronously, so the rest of the page should experience no delay in rendering.
